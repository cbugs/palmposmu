Favicon Package
Generated by 3utils.com

This package contains favicons in multiple sizes:
- favicon.ico: Standard ICO format (32x32) for maximum browser compatibility
- favicon-16x16.png: Standard favicon size
- favicon-32x32.png: Standard favicon size
- favicon-48x48.png: For high-DPI displays
- favicon-64x64.png: For Windows tiles
- favicon-128x128.png: For Chrome Web Store
- favicon-256x256.png: For larger displays and future use

How to use:
1. Upload all favicon files to your website's root directory
2. Add the following to your HTML <head>:

<link rel="icon" type="image/x-icon" href="/favicon.ico">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="48x48" href="/favicon-48x48.png">

All images preserve transparency from your original image.
